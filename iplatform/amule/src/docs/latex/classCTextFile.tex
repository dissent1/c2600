\subsection{Detailed Description}
Text file class. This class is a wrapper around wxFFile, letting an text file be read or written line-\/by-\/line. The class provides transparent and automatic EOL-\/style handling.

Note that it is not possible to seek in a \doxyref{CTextFile}{p.}{classCTextFile}, only sequential reading or writing is possible. Also note that the maximum length of a line is fixed (see \doxyref{CTextFile::GetNextLine}{p.}{classCTextFile_a33df174e11ab1ed87163ee33281caae8}), however this shouldn't be a problem, given the uses of this class.
