\subsection{Detailed Description}
Simple tree item. This tree item has one value and nothing speciality. :) The value might be an arbitrary integer or floating point type, or a wxString string.

The item is able to display value in different formats, see \doxyref{SetDisplayMode()}{p.}{classCStatTreeItemSimple_a5f0a79371e2372c966cccd805a0b8c88}.

\begin{DoxyNote}{Note}
that you have to specify the right format code on 'label', i.e.: s for string and integers with displayMode dmTime or dmBytes, u or similar for integers, and f or similar for floating point types.

You have to call \doxyref{SetValue()}{p.}{classCStatTreeItemSimple_a341be2daf767e3054dd723722749a72b} after creation for non-\/integer values, otherwise you'll get undesired results. 
\end{DoxyNote}
