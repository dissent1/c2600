\subsection{Detailed Description}
\doxyref{MuleGifCtrl}{p.}{classMuleGifCtrl} is a simple widget for displaying a gif animation. It is based on the animation classes by Julian Smart and Guillermo Rodriguez Garcia, but is specialized for the reduced requirements of the aMule project. It provides flicker-\/free redrawing using wxBufferedPaintDC.

To reduce complexity, several things have been hardcoded, though they can easily be changed:
\begin{DoxyItemize}
\item The animation will continue to loop until \doxyref{Stop()}{p.}{classMuleGifCtrl_a6f9f7ad5456fabe152a7bc7f5bcb644b} is called.
\item The gif image is assumed to be transparent.
\item Start will start the animation from the first frame and wont continue a stopped animation. 
\end{DoxyItemize}
