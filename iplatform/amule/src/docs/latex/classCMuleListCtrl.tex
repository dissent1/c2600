\subsection{Detailed Description}
Enhanced wxListCtrl provided custom-\/drawing among other things. This class provides these features which the original wxListCtrl lacks:
\begin{DoxyItemize}
\item Automatic sort arrows upon clicks on the column headers
\item Custom drawing of items.
\item Hiding of columns through auto-\/generated popup-\/menu.
\item Helper function for inserting items pre-\/sorted.
\item Loading and saving of column properties.
\item Selection of items by typing an initial part of the text (TTS). 
\end{DoxyItemize}
