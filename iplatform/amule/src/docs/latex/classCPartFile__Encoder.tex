\subsection{Detailed Description}
PartStatus strings and gap lists are quite long -\/ RLE encoding will help.

Instead of sending each time full part-\/status string, send RLE encoded difference from previous one.

\doxyref{PartFileEncoderData}{p.}{classPartFileEncoderData} class is used for decode only, while \doxyref{CPartFile\_\-Encoder}{p.}{classCPartFile__Encoder} is used for encode only.
