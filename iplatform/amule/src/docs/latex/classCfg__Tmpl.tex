\subsection{Detailed Description}
\subsubsection*{template$<$typename TYPE$>$ class Cfg\_\-Tmpl$<$ TYPE $>$}

Template Cfg class for connecting with widgets. This template provides the base functionionality needed to syncronize a variable with a widget. However, please note that wxGenericValidator only supports a few types (int, wxString, bool and wxArrayInt), so this template can't always be used directly.

\doxyref{Cfg\_\-Str}{p.}{classCfg__Str} and \doxyref{Cfg\_\-Bool}{p.}{classCfg__Bool} are able to use this template directly, whereas \doxyref{Cfg\_\-Int}{p.}{classCfg__Int} makes use of serveral workaround to enable it to be used with integers other than int.
