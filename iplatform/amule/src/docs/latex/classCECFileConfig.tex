\subsection{Detailed Description}
Extension to wxFileConfig for reading/writing \doxyref{CMD4Hash}{p.}{classCMD4Hash} values. This class converts between the text and binary representation of an MD4 hash, and also maps empty strings to the empty hash and vica versa.
