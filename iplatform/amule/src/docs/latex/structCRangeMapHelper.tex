\subsection{Detailed Description}
\subsubsection*{template$<$typename VALUE, typename KEYTYPE$>$ struct CRangeMapHelper$<$ VALUE, KEYTYPE $>$}

Default helper structure for normal \doxyref{CRangeMap}{p.}{classCRangeMap} instantations. Specializations should must have the following properties.
\begin{DoxyItemize}
\item The four value typedefs (see comments for details).
\item A template-\/defined member variable named 'first'.
\item A comparison operator that doesn't consider the 'first' field.
\end{DoxyItemize}

The typedefs are used to specify the return values of iterators.
